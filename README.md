## Project Overview

This SQL project analyses UK railway journey data to explore delays, revenue, and operational patterns.  
The goal is to use SQL to uncover insights that could help railway operators understand performance and customer behaviour.

---

## Business Questions

This analysis answers the following questions:

1. Which routes experience the highest average delays?

2. What are the most common reasons for train delays?

3. Do longer delays lead to more refund requests?

4. Which stations handle the highest number of journeys?

5. Which routes generate the most revenue?

---

## Key Insights

- Certain routes consistently experience higher average delays than others.
- Some delay causes appear far more frequently than others.
- Refund requests tend to occur on journeys with longer delays.
- A small number of stations handle a large proportion of journeys.
- Certain routes generate significantly higher ticket revenue.
